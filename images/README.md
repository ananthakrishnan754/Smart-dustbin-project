# Project Showcase Images

This directory hosts the media showcase files referenced by the primary repository documentation.

To ensure the images display perfectly inside the main `README.md` preview, save or drag-and-drop your screenshots into this directory matching the following target filenames:

- `hardware_prototype.jpg` (or `.png`) - The physical smart dustbin box prototype showcasing the automated lid and interior sorting flaps.
- `train_admin_dashboard.png` - The full web management UI view illustrating targeted broadcast functionalities.
- `worker_and_station_admin.png` - The split view showcasing synchronized leaderboard points and live sensor fill progress.
